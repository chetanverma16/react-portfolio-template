2. Data preparation
  • Data Storage: sql or no sql database,database concurrency
    1. structured data (categorical, numerical) ‚Üí postsql, mysql
    1. unstructured data (text, video, image) ‚Üímongo DB
  • Feature engineering: feature scaling
    • missing value handling:
      1. imputation
        1. median/mean/mode filling
        1. default value filling
        1. CON: cause some biases or noises to the model
      1. Deletion
        1. row deletion (when record miss too many value)
        1. col deletion (when feature miss too many value)
        1. CON: reduce too many records may harm model performance
    • Feature Scaling
      1. Numerical data
        1. normalization (min-max): z = (x - x_min) / x_max - x_min
          1. does not change distribution, make feature in range [0,1]
        1. standardization (z-score): z = x-mean / sd
          1. changing distribution to mean 0, sd 1
        1. log scaling: log(x),
          1. make distribution less skewed, converge faster
      1. Discretization: convert continuous to categorical feature
        1. bucketing: e.g. let age 1-10 become group 1, age 11-20 become group 2‚Ä¶
      1. Categorical data:
        1. integer encoding: e.g. 1 for good, 0 for bad
          1. CON: when relationship is not ordinal, this is not good idea
        1. one hot encoding: convert to unique binary feature e.g. red 001 blue 010 green 100
          1. CON: when unique feature num is large
        1. embedding: project feature to N Dimension vector
          1. PRO: when unique feature num is large
    1. text, image: data augmentation

Talking points:
  • Data availability?
    • what are the data sources?
    • what data available
    • how do we collect data
    • how large is the data size
    • how often the new data come in
  • Data Storage?
    • where is data stored?
    • is it on cloud or local?
    • which data format is appropriate for stroing data?
    • how do we store multimodal data?

  • Feature Engineering:
    • how do we process raw data into useful form for model?
    • how should we handle missing value?
    • is feature engineering required?
    • do we need to normalize data?
    • which feature that we should construct from raw data?
    • how do we combine multimodal data?
  • Privacy?
    • how sensitive is the data?
    • are user concern about privacy?
    • is anonymization of user data useful?
    • is it possible to store user data in cloud or just user devices?
  • Bias?
    • are there any biases in data?
    • what are they and how should we correct them?
